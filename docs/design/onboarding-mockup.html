THE APPROACH — built for a crew that has never used an app like this

SHOW, DON'T TELL. The payoff (a real estimate) appears on screen 1; the three "how it works" beats each show a phone doing the thing, not a paragraph explaining it.  •  PLAIN WORDS. No "AI", "transcript", "extraction", "record". Just: talk · notes · fix · paperwork · send.  •  ANSWER THE FEAR. Screen 3 ("first draft — you're in control") tackles the non-technical user's real worry — *what if it's wrong?* — head-on.  •  SET UP THE MINIMUM. Only name/trade/mic; branding, contact, terms all come later in the PAPER tab, so setup is 30 seconds.  •  LEARN BY DOING. It ends on the real board with one coach mark on START WALK — no multi-step tour.  •  SKIPPABLE. A returning user or someone impatient taps SKIP straight to setup.

OPEN QUESTIONS FOR ISAAC: (a) intro = 3 beats (shown) or fewer? (b) add an optional guided practice walk (run the demo with narration) for the truly hesitant, or is the coach mark enough? (c) should coach marks also tag DONE + "tap to fix" on the first real walk, or just START?
